---
title: The corral (canvas view)
description: The main canvas where agents, tasks, and claims show up as they happen.
---

The corral is the default tab: every active agent renders as a moving node,
connected by a line to each file path it currently holds a claim on. Click
any agent — on the canvas, or in the roster on the right — to open its
floating detail window: what it's working on, which paths it holds, its
recent activity, and the memory entries it can recall.
