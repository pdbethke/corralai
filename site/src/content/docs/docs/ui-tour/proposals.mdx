---
title: Proposals
description: The tab where the learning loop's proposed skills and lessons wait for review.
---

The Proposals tab shows every pending skill proposal, with the live count
badge that also appears on the tab itself. Approving here (or via
`corral-admin proposals approve <id>`) promotes the proposal's guidance into
vetted memory and syncs the matching skill fleet-wide — see [the learning
loop](/docs/concepts/memory-and-learning-loop/). Rejecting sends the pattern
back until the evidence doubles. This is the human gate of the learning loop.
