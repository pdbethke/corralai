The Topology tab is the corral's ecosystem map: every host machine, the
agents running on it, and — for each agent — the model backing it and the
exec-isolation jail confining its commands. It's observability, not control:
useful for seeing at a glance which host is carrying the load, or whether an
agent is running with network access it shouldn't need.
